# PyRadioPlayer

"PyRadioPlayer" is a internet radio player with Python. Mainly created by ChatGPT.

# Requirement

* Python 3.10.9
* PySimpleGUI 4.60.4
* pynput 1.7.6
* python-vlc 3.0.18121
* streamlink 5.4.0

Tested on Windows11 only.
